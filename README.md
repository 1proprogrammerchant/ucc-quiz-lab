# UCC Quiz Lab

Playful, animated quiz site for the University of Commonwealth Caribbean. This project was personal and educational, it is not to be taken seriously. It features a fun, interactive quiz experience with a focus on campus life and computer science basics.

Built by **Me** (GitHub: [1proprogrammerchant](https://github.com/1proprogrammerchant))

## Features

- **60 questions** across 2 tracks: On just CS Basics(for now)
- **Advanced scoring**: Percentage, letter grades, star ratings, streak bonuses
- **Smooth animations**: Fade-in reveals, floating orbs, theme switcher(removed will add a new one soon)
- **Live stats**: Real-time streak, accuracy, and bonus multipliers
- **Quiz controls**: Shuffle, track filters, skip questions
