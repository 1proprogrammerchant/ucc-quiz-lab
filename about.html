About UCC Quiz Lab

A modern, high-performance quiz platform designed to make learning interactive, engaging, and fun.

info

Our Mission

UCC Quiz Lab was built to transform how students approach learning. We believe education should be engaging, rewarding, and accessible to everyone. Our platform gamifies the learning process with streak tracking, real-time scoring feedback, and beautiful UI/UX design.

The Technology

html

Frontend

Modern HTML5, CSS3, and vanilla JavaScript (ES6 modules). No build tools, no bloat-just clean, performant code.

memory

C++ WebAssembly

The scoring engine is written in C++ and compiled to WebAssembly via Emscripten for blazing-fast calculations.

dns

Backend

Python HTTP server for local development. Easily deployable via Docker and nginx for production environments.

database

Database

Browser local storage for score persistence. Future versions will include cloud sync and leaderboards.

palette

Design System

Custom CSS with CSS variables for theming. Two built-in themes: Default dark and Sunset purple.

auto_awesome

Animation

Smooth transitions and Intersection Observer-based reveal animations create an engaging user experience.
